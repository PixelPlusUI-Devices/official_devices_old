# Installation Guide For PixelPlusUI on Olivewood/lite

### CLEAN FLASH :
1. Format Data
2. Wipe Cache, Dalvik, System, Vendor
3. Flash ROM
4. Reboot

### DIRTY FLASH IF AN UPDATE
1. Wipe Dalvik and Cache
2. Flash ROM
3. Reboot 

## Note; 
* Coming from Ports or HentaiOS = Format Data
